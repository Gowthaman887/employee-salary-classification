A completely serverless Machine Learning project that classifies employee salaries into **Low**, **Medium**, or **High** bands using **Logistic Regression**. 

What makes this project unique? The Python ML model automatically trains, calculates metrics (precision, recall, F1, confusion matrix), and seamlessly **embeds the results directly into an interactive HTML dashboard** — no backend API, no database, no servers required to view the results! Just open `index.html` in any browser.

## 🚀 Features

- **Machine Learning Pipeline**: Uses `scikit-learn` with feature engineering (interaction & ratio features), `StandardScaler`, and `PolynomialFeatures` to capture non-linear relationships before passing them to a Multinomial Logistic Regression model.
- **Dynamic Classification**: Automatically categorizes raw numerical salary data:
  - **Low**: `< 100,000`
  - **Medium**: `100,000 - 999,999`
  - **High**: `>= 1,000,000`
- **Serverless Analytics Dashboard**: A single-page, beautifully styled responsive UI using vanilla CSS and Chart.js.
- **Automated Reporting**: Running the Python script automatically updates `results.json` and dynamically patches `index.html` with the latest model inferences.

## 🛠️ Tech Stack

- **Data Processing & ML**: `pandas`, `numpy`, `scikit-learn`
- **Frontend Dashboard**: HTML5, Vanilla CSS, `Chart.js` via CDN
- **Dataset**: Custom `Employee_Salary_Dataset` (Available in CSV & ODS)

## 🧠 How the Model Works

1. **Feature Engineering**: The model extracts `Age`, `Experience_Years`, and encoded `Gender`. We add engineered interaction features (`Exp_x_Age`) and ratio features (`Exp_per_Age`) to better separate the salary classes.
2. **Pipeline**: The data pushes through:
   - `StandardScaler()` to normalize feature variances.
   - `PolynomialFeatures(degree=2)` to map non-linear dimensions.
   - `LogisticRegression(multinomial)` to output the final 3-class probability.
3. **Payload Construction**: The code dumps evaluation numbers (Test/CV Accuracy, Classification Report, Confusion Matrix, row-by-row inferences) into a JSON dictionary and injects it directly into the JavaScript of `index.html`.

## ⚙️ How to Run Locally

Since the `index.html` file is pre-populated with data, you can simply **double-click `index.html`** right now to view the dashboard! 

To retrain the model or test changes to the Python code:

### 1. Prerequisites
Ensure you have Python installed. Then, install the required packages:
```bash
pip install pandas numpy scikit-learn
```

### 2. Run the Model
Execute the main script. It will train the model, save the results, and update the dashboard:
```bash
python model.py
```

### 3. View the Results
Open the freshly updated dashboard in your web browser:
- **Windows**: `start index.html`
- **Mac**: `open index.html`
- **Linux**: `xdg-open index.html`
